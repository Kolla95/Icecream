# 🍦 Sunny Scoops – Ice Cream Shop Website

Sunny Scoops is a pastel-inspired ice cream shop website designed to showcase clean UI, playful branding, and a friendly small-business feel.

The project focuses on layout, color systems, and user experience rather than complex functionality, creating a visually engaging and approachable website.

## ✨ Features

- Clean, structured multi-section layout
- Pastel color palette and cohesive branding
- Featured products and categorized menu sections
- Clear call-to-action buttons (order, menu, visit)
- Responsive and user-friendly design

## 🛠️ Tech Stack

- HTML5
- CSS3

## 🎯 Purpose

This project was built to practice:
- UI/UX design principles
- Branding and visual consistency
- Layout structuring for small business websites
- Creating visually appealing, portfolio-ready designs

## 🚀 Future Improvements

- Add ordering system functionality
- Add animations and micro-interactions
- Improve mobile responsiveness
- Connect to a real backend or ordering API
